# Paradise Nursery Shopping Application

## Project Overview
This project is a dynamic web application built using **React** and **Redux**. It allows users to browse a variety of house plants, add them to a shopping cart, and manage their cart items efficiently. The application focuses on providing a user-friendly e-commerce experience for an online plant shop.

## Key Features
- **Product Listing**: Displays plants grouped into categories (Indoor, Outdoor, Succulents, etc.) with thumbnails, names, and prices.
- **Interactive Shopping Cart**: 
    - Add items to the cart with a single click.
    - Automatic button disablement once a product is added.
    - Dynamic cart icon showing the total number of items.
- **Cart Management**: 
    - View all added plants in the cart.
    - Increase or decrease item quantities.
    - Remove items from the cart.
    - Automatic calculation of total cost per item and total cart amount.
- **Navigation**: Intuitive navigation bar to switch between the Home, Plants, and Cart pages.

## Technologies Used
- React.js
- Redux Toolkit (for state management)
- CSS3 (for styling)

## How to Run
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the application.
